# [144. Binary Tree Preorder Traversal](https://leetcode.com/problems/binary-tree-preorder-traversal/)


## 题目

Given a binary tree, return the preorder traversal of its nodes' values.

Example
```c
Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [1,2,3]
```

## 题目大意

给定一个二叉树，返回他的前序遍历

## 解题思路

前序遍历 根->左->右
